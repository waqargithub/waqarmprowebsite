Ruby on Rails Project: Web Application to Create User Accounts

Back to top

This project creates a web application that allows creating an account that generates an account activation email. The user may then modify their profile and generate password reset requests. The application allows viewing a list of other users but ensures that only authorized are users may modify or delete an account. This project involved cloud-based linux development and deployment, automated tests, bitbucket (similar to github), and application deployment on heroku. Future iterations will support generating microblogs (similar to tweets) and the ability to follow other users' activity.
